## Описание:
Данный проект является мои первым пет-проектом и имеет версию 1.0.

Идея проекта представляет собой систему, веб-приложение работающее с бд кошельков, которое имеет ряд методов и сервлетов, таких как:
- Пополнение счета кошелька;
- Обмен валют;
- Обновление кошелька;
- Добавление кошелька;
- Удаление кошелька.

Проект имеет проработанную (не полностью) логику работы и связи сервлетов с базой данных и страницами, свою систему проверки
на ввод ошибочный данных, пустых строк.

Данная версия проекта является базированной, т.е проект будет иметь более новые версии, с новыми доработками, подключенными
фреймворками, новыми реализациями, информацию насчёт обновлений можно почитать в [TODO.md](TODO.md)

## Использованные технологии:
