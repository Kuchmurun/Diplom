## Описание:

Данный проект является обновленной версией пет проекта переведённую на версию 2.0 (сентябрь 2025).

## Нововведения:

- Проект был полностью переведен на архитектуру Spring.
- Добавлен Spring Boot.
- Добавлен Spring MVC.
- Глобальная переработка проекта, разбиения проекта на независимые части по правилу внедрения зависимостей в Spring.
- Внедрение автоматических генераций многих компонентов, сокращение объема кода, увеличение функционала.
- Добавлена валидация, переработана система обработки ошибок.
- Переработана логика извлечения названия валюты и курса перевода.
- Переход с JSP страниц на HTML.
- Структура проекта переведена на продакшен-стиль.
- Переход на MapStruct для маппинга DTO - Entity.

Идея проекта представляет собой систему, веб-администрирование веб-кошельков работающее с бд кошельков,
которое имеет ряд функций, таких как:

- Пополнение счета кошелька;
- Обмен валют;
- Обновление кошелька;
- Добавление кошелька;
- Удаление кошелька;
- Валидация и проверка ошибок.

Проект имеет проработанную (не до конца) логику работы и связи веб-приложения с базой данных и страницами, свою систему
проверки на ввод ошибочных данных, пустых строк.

Данная версия проекта является обновленное, проект также будет переведен на новую версию, с новыми доработками,
подключенными
фреймворками, новыми реализациями, информацию насчёт обновлений можно почитать в [TODO.md](TODO.md)
